## Technologies Used

- Next.js
- TailwindCSS
- TypeScript
- Shadcn/UI
- Vercel
- Directus
- Digitalocean
- Turborepo

## Prerequisites

- Node.js
- pnpm

The cms directory will need an .env file for directus to work.

## How to run

Run the following command to start the the nextjs app and the directus server.

```bash
pnpm dev
```
